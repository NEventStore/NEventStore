NEventStore

A persistence agnostic Event Store for .NET

Download this project as a .zip file Download this project as a tar.gz file

Overview

The NEventStore is a persistence library used to abstract different storage implementations when using event sourcing as storage mechanism. Event sourcing is most closely associated with a concept known as CQRS.

Start using NEventStore

The most recent stable release is avaiable on NuGet.org.
